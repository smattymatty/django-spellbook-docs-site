## Multiple Sources Configuration

This example project uses multiple sources to demonstrate how to organize different types of content:

```python
# settings.py configuration for this example project
SPELLBOOK_MD_PATH = [BASE_DIR / "docs", BASE_DIR / "blog"]
SPELLBOOK_MD_APP = ["docs_app", "blog_app"]
SPELLBOOK_MD_URL_PREFIX = ["docs", "blog"]
SPELLBOOK_MD_BASE_TEMPLATE = ["cosmic/docs_base.html", "cosmic/blog_base.html"]

{~ alert type="info" ~}
The documentation is configured to use /docs/ as its URL prefix, while the blog uses /blog/ as its prefix.
```
